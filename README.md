# Caro Aesthetics Clinic

Site vitrine Vite + React prêt pour Vercel.

## Modifier les images

Les images du logo, du hero et de la galerie se modifient dans :

```txt
src/data/siteContent.ts
```

Vous pouvez utiliser :

- une URL directe d'image ;
- un lien Google Drive partagé au format `https://drive.google.com/file/d/.../view`.

Pour Google Drive, le fichier doit être partagé avec l'option "Toute personne disposant du lien".

## Déploiement Vercel

- Build command : `npm run build`
- Output directory : `dist`
- Install command : `npm install`
